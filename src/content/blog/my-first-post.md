## 🚀 欢迎来到我的数字花园！

这是我使用 **Astro** 搭配 **Tailwind CSS** 搭建的全新个人博客。整个搭建过程非常顺利，成功解决了一些环境配置和路径问题，最终让它完美上线了。

### 为什么选择这个组合？

1. **极速体验**：Astro 默认生成的都是静态 HTML，几乎没有多余的 JavaScript，打开速度飞快。
2. **样式自由**：Tailwind CSS 让页面样式的调整变得极其直观和现代。
3. **完全免费**：托管在 GitHub Pages 上，不需要花一分钱服务器费用。
